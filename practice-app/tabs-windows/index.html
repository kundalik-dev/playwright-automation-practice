Tabs & Windows — Playwright & Selenium Practice

Practice opening, switching, and closing tabs/windows. Playwright key API: page.waitForEvent('popup'), context.pages(). Selenium key API: getWindowHandles(), switchTo().window(handle).

1. Anchor target="_blank" Beginner

Playwright: await Promise.all([page.waitForEvent('popup'), link.click()]) — the promise must be set up BEFORE the click or you'll miss the event. Selenium: save current handle → click → diff getWindowHandles() for the new one.

3. Named Window — Reuse Behavior Intermediate

window.open(url, 'myWindow'): if a window named myWindow is already open, the browser reuses it (navigates) instead of opening a new one. Interview Q: "Will clicking twice open two popups?" — Answer: No, only one named window exists. Playwright: context.pages() will show just one extra page after two clicks.

4. Multiple Windows Management Intermediate

Playwright: const pages = context.pages() returns all open pages as an array. Selenium: Set<String> handles = driver.getWindowHandles() — iterate with a loop. Practice: open 3 windows, switch to each, assert title/URL, close Window 2, verify only 2 remain.

    7. Playwright vs Selenium — Cheat Sheet

    Action Playwright (JS) Selenium (Java)
    Intercept new tab / popup page.waitForEvent('popup') — set up BEFORE click Store handles before click → diff after
    Get all open pages context.pages() driver.getWindowHandles()
    Switch to page / window Use the returned Page object directly (no switch needed) driver.switchTo().window(handle)
    Get opener of popup await popup.opener() — returns parent Page No direct API; switch back by stored handle
    Close specific tab await popup.close() Switch to it then driver.close(), then switch back
    Wait for popup to close await popup.waitForEvent('close') Poll getWindowHandles().size() until it decreases
    Get URL / title of popup popup.url() / await popup.title() Switch first → driver.getCurrentUrl() / getTitle()
    Context-level page listener context.on('page', async page => { … }) No direct equivalent — must poll handles

    ⭐ Playwright tip: waitForEvent('popup') must be awaited BEFORE the action that opens the window — Promise.all() pattern prevents race conditions.
